# UIML 文档库

UIML v2 （由 RigoLigo 重构并维护）使用基于[PEPB (GitHub)](https://github.com/RigoLigoRLC/TarsGo_PEPB/)的嵌入式工具链进行编译。在开始前，请先阅读 PEPB 的开发文档，熟悉如何使用 PEPB 编译、下载、调试 STM32 工程，以及熟悉基础的 CMakeLists 编写方式。
